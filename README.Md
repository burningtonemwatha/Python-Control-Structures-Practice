## Python Practice ‚Äì Conditional Statements, Data Types & Loops

This project contains **15 small Python programs** that demonstrate the use of **conditional statements**, **loops**, **data types**, and **basic collection operations**.
Each file is written as a **stand-alone script**, meaning you can run them individually to test your understanding of Python control structures.

---

### Learning Objectives

* Practice using `if`, `elif`, and `else` statements
* Work with `for` and `while` loops
* Understand `match‚Äìcase` (Python 3.10+)
* Perform simple type conversions and dictionary operations
* Use collections like lists and dictionaries effectively

### How to Run

1. Make sure you have **Python 3.10 or later** installed.
2. Download or clone this repository.
3. Open a terminal or code editor (like VS Code or IDLE).
4. Run any file with:

   ```bash
   python Q1_AgeAccess.py
   ```

   *(Replace the file name with the one you want to test.)*
